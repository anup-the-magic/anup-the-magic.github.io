Pro-Bending

An Avatar The Last Airbender-inspired Arena fighting game

In seeking to recreate the dynamic, immersive bending battles from The Legend of Korra, I wanted to create a full end-to-end game to to apply everything I've learned.

As a solo project which I was passionate about, I was mindful of keeping the feature-set constrained -- I wanted to make sure that the full experience was fleshed out before additional breadth was added. Despite this focus, I was able to ensure that the project always felt production-ready, planning for features I knew would be necessary down the line and keeping the features constrained. With a concrete end goal always at the forefront, I was able to build out a foundation for a multiplayer arena fighting game in a little under 3 months of dedicated dev work.
